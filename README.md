FlowForward 是一款基于命令行实现的功能强大的TCP流量转发工具，用于在后渗透中的横向越权时使用，该工具可定向转发数据包，打破内外网的限制，实现对内网特定主机发起攻击或中转数据包到外网，工具只有19KB绝对小巧且无任何特征，是一款不错的渗透辅助工具。
